# Agent监控后台使用指南

## 访问方式

### 方法1：侧边栏导航（推荐）
1. 使用管理员账号登录（用户名：`xiang`）
2. 在左侧侧边栏中，您会看到：
   - 智能对话
   - 数据记录
   - 数据分析
   - **--- 分隔线 ---**
   - **Agent监控** ⭐（管理员专属）

3. 点击"Agent监控"即可进入

### 方法2：用户菜单
1. 点击左下角的用户头像
2. 在下拉菜单中选择"Agent监控"

### 方法3：直接访问URL
在浏览器地址栏输入：`/admin`

## 功能说明

### 统计卡片
- **总调用次数**：显示AI对话的总次数和成功率
- **平均处理时间**：Memory Update处理的平均耗时
- **数据记录**：自动记录的血糖、饮食、运动数据总数
- **最活跃Agent**：调用次数最多的Agent

### 调用日志Tab
查看最近100条AI对话记录，包括：
- 用户消息和AI回复
- 涉及的Agent（彩色标签）
- Memory Update处理结果
- 数据记录情况
- 处理时间和状态

### Agent统计Tab
查看各Agent的调用频率：
- 对话协调Agent（coordinator）
- 数据记录Agent（data_recorder）
- 即时分析Agent（analyzer）
- 教育科普Agent（educator）
- 激励支持Agent（motivator）

### 工作流分析Tab
了解多Agent协同工作流程：
- 典型的5步工作流程
- Memory Update处理统计
- 数据记录效率分析

## 管理员账号

第一个注册的用户自动成为管理员。当前管理员账号：
- 用户名：`xiang`
- 角色：admin

## 注意事项

1. **仅管理员可见**：普通用户看不到"Agent监控"入口
2. **实时更新**：每次AI对话后自动记录日志
3. **权限控制**：普通用户只能查看自己的日志，管理员可查看所有日志

## 如果找不到入口

请确认：
1. ✅ 您已登录管理员账号（用户名：xiang）
2. ✅ 页面已刷新（Ctrl+R 或 Cmd+R）
3. ✅ 查看左侧侧边栏是否有"Agent监控"选项
4. ✅ 如果在移动端，点击左上角菜单图标查看

## 测试方法

1. 登录管理员账号
2. 在"智能对话"页面发送几条消息
3. 进入"Agent监控"页面
4. 查看"调用日志"Tab，应该能看到刚才的对话记录
5. 查看"Agent统计"Tab，了解各Agent的调用情况

## 技术支持

如果仍然无法访问，请检查：
- 浏览器控制台是否有错误信息
- 网络连接是否正常
